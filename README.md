## Usage

First, add your `SESSION_SECRET` to your `.env` file:

```sh
SESSION_SECRET=some_secret
```

Then you can install your dependencies and run the server with these commands:

```sh
npm install
git submodule update --init
node main.js
```

## Dependencies

```
Node.js
```
